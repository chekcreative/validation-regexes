## Uppercase Letter
Validate that a string contains at least one uppercase letter.
```
/^(.*[A-Z].*)$/
```

## Lowercase Letter
Validate that a string contains at least one lowercase letter.
```
/^(.*[a-z].*)$/
```

## Special Character
Validate that a string contains at least one special character.
```
/^(.*[a-z].*)$/
```
